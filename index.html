Gem Collector

You will be given a random number to start the game.
There are four crystals, each with a different random number assigned to it, click on them to add to your score.
The goal is to match the number exactly, giving yourself a win. If you go over you will be given a loss.
A new random number will be given once the game is over. This will also reset the value of each crystal. Good luck and have fun.
